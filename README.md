# College Schedule Mobile

Мобильное приложение для просмотра расписания занятий колледжа

## Технологии
- Kotlin
- Jetpack Compose
- Material Design 3
- Retrofit
- DataStore

## Структура проекта
- **data** - работа с данными (API, DTO, Repository)
- **ui** - пользовательский интерфейс (Compose)
- **utils** - вспомогательные утилиты

## Требования
- Android Studio Hedgehog+
- Minimum SDK: API 31 (Android 12)
- Kotlin 1.9+
